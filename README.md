# KHHouse 高雄重點行政區建案儀表板

涵蓋高雄市 9 個重點行政區（楠梓、仁武、左營、鼓山、三民、新興、前金、苓雅、鳳山）的建案資訊網站。

- 前台：靜態 `data.json` ＋ HTML/JS 儀表板（沿用 [NKUinfos](https://github.com/normalsky315-jpg/NKUinfos) 架構，新增行政區篩選）
- 資料品質：由 [HouseQA](../HouseQA) 做 591 / house958 / 樂居多來源交叉比對＋ AI 差異分析
- 分批上線：Tier 1（楠梓、仁武）→ Tier 2（左營、鼓山、三民）→ Tier 3（新興、前金、苓雅）→ Tier 4（鳳山）
