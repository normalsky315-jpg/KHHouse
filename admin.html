輸入 GitHub Token 後即可讀取並編輯網站資料。Token 只儲存在你這台裝置的瀏覽器中。
第一次使用？Token 申請步驟（約 5 分鐘）
1. 登入 GitHub → 右上頭像 → Settings
2. 左欄最底 Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
3. 名稱隨意（如 NKU後台）、Expiration 選 90 days 或 custom 一年
4. Repository access 選 Only select repositories → 勾 NKUinfos
5. Permissions → Repository permissions → Contents 改成 Read and write
6. 按 Generate，把 github_pat_… 整串複製貼到上面欄位
※ Token 等同你的鑰匙，不要貼給任何人、不要存在共用電腦。
